# Buchungsnotiz hinzufügen

## User Story

Als INNOQ-Mitarbeiter möchte ich eine Buchungsnotiz hinzufügen, damit ich zusätzliche Informationen zur Buchung hinterlegen kann.

## Beschreibung

Bei der Buchung eines Konferenzraums gibt es häufig Informationen, die über den Meetingtitel hinausgehen und für die Buchung relevant sind. Ein INNOQ-Mitarbeiter möchte beispielsweise vermerken, dass besonderes Equipment benötigt wird, dass externe Gäste erwartet werden, oder spezielle Hinweise für andere Kollegen hinterlassen. Die Buchungsnotiz ist ein optionales Freitextfeld, das dem Mitarbeiter erlaubt, diese zusätzlichen Kontextinformationen zu seiner Raumbuchung zu dokumentieren.

Diese Funktion unterstützt das Bedürfnis nach Transparenz, indem relevante Details zur Buchung für alle Beteiligten sichtbar werden. Die Notiz kann auch bei späteren Rückfragen oder zur eigenen Erinnerung hilfreich sein.

## Akzeptanzkriterien

- [ ] Ein optionales Freitextfeld für die Buchungsnotiz wird im Buchungsformular angezeigt
- [ ] Die Buchungsnotiz kann bis zu 500 Zeichen umfassen
- [ ] Die Buchung kann auch ohne Eingabe einer Notiz abgeschlossen werden
- [ ] Die eingegebene Buchungsnotiz wird mit der Raumbuchung gespeichert
- [ ] Die Buchungsnotiz ist in der Buchungsübersicht sichtbar
- [ ] Sonderzeichen und Zeilenumbrüche werden korrekt verarbeitet
