# Buchungsdetails ansehen

## User Story

Als INNOQ-Mitarbeiter möchte ich eine einzelne Raumbuchung im Detail ansehen, damit ich alle relevanten Informationen zu einer Buchung abrufen kann.

## Beschreibung

Ein INNOQ-Mitarbeiter hat bereits eine oder mehrere Raumbuchungen vorgenommen und möchte nun die vollständigen Details einer bestimmten Buchung einsehen. Aus der Buchungsübersicht heraus kann der Mitarbeiter eine einzelne Buchung auswählen, um alle relevanten Informationen auf einen Blick zu erhalten. Dies umfasst den gebuchten Konferenzraum mit seiner Ausstattung, den genauen Zeitraum, den Standort sowie ggf. den Buchungszweck. Die Detailansicht dient als zentraler Ausgangspunkt für weitere Aktionen wie das Exportieren, Teilen, Stornieren oder Ändern der Buchung.

## Akzeptanzkriterien

- [ ] Aus der Buchungsübersicht kann eine einzelne Buchung zur Detailansicht ausgewählt werden
- [ ] Der Name des gebuchten Konferenzraums wird angezeigt
- [ ] Der Standort des Konferenzraums wird angezeigt
- [ ] Das Buchungsdatum wird angezeigt
- [ ] Die Buchungszeit (Start- und Endzeit) wird angezeigt
- [ ] Die Ausstattung des Konferenzraums wird angezeigt
- [ ] Der Buchungszweck wird angezeigt (sofern bei der Buchung angegeben)
- [ ] Es gibt eine Möglichkeit, zur Buchungsübersicht zurückzukehren
- [ ] Von der Detailansicht aus sind Aktionen wie Exportieren, Teilen, Stornieren und Ändern erreichbar
