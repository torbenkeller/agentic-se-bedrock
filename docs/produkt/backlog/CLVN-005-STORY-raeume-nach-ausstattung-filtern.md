# Räume nach Ausstattung filtern

## User Story

Als INNOQ-Mitarbeiter möchte ich Räume nach Ausstattung filtern, damit ich einen Raum mit der benötigten technischen Ausstattung finde.

## Beschreibung

INNOQ-Mitarbeiter benötigen für unterschiedliche Anlässe unterschiedliche technische Ausstattung in Konferenzräumen. Ein Kundenworkshop erfordert möglicherweise Videokonferenz-Equipment und ein großes Whiteboard, während ein internes Team-Meeting nur einen Bildschirm für Präsentationen benötigt.

Diese User Story ermöglicht es Mitarbeitern, die Raumliste nach verfügbarer Ausstattung zu filtern. So finden sie schnell einen Konferenzraum, der alle benötigten Ausstattungsmerkmale bietet, ohne jeden Raum einzeln prüfen zu müssen.

Die Filterfunktion ergänzt die Standortauswahl (CLVN-002), die Raumübersicht (CLVN-003) und den Kapazitätsfilter (CLVN-004) zu einem vollständigen Suchworkflow für die passende Umgebung.

## Akzeptanzkriterien

- [ ] Eine Liste der filterbaren Ausstattungsmerkmale wird angezeigt (z.B. Bildschirm, Whiteboard, Videokonferenz-Equipment)
- [ ] Mehrere Ausstattungsmerkmale können gleichzeitig ausgewählt werden
- [ ] Nach Auswahl von Ausstattungsmerkmalen werden nur Konferenzräume angezeigt, die alle ausgewählten Merkmale besitzen
- [ ] Die Anzahl der gefundenen Räume wird nach dem Filtern aktualisiert
- [ ] Der Ausstattungsfilter kann mit dem Kapazitätsfilter kombiniert werden
- [ ] Die aktiven Filter sind für den Nutzer sichtbar und können einzeln entfernt werden
- [ ] Wenn keine Räume den Filterkriterien entsprechen, wird eine entsprechende Meldung angezeigt
