# Arbeitsplatz auswählen

## User Story

Als INNOQ-Mitarbeiter möchte ich einen verfügbaren Arbeitsplatz auswählen und bestätigen, damit ich den für mich passenden Platz reservieren kann.

## Beschreibung

Ein INNOQ-Mitarbeiter plant einen Bürotag und möchte vorab sicherstellen, dass ein geeigneter Arbeitsplatz verfügbar ist. Die Anwendung zeigt alle verfügbaren Arbeitsplätze für das gewählte Datum an. Der Mitarbeiter kann einen Platz auswählen, Details einsehen (z.B. Ausstattung, Lage) und seine Auswahl bestätigen.

## Akzeptanzkriterien

- [ ] Verfügbare Arbeitsplätze für ein gewähltes Datum werden angezeigt
- [ ] Bereits gebuchte Arbeitsplätze sind als belegt gekennzeichnet
- [ ] Arbeitsplatz-Details (Ausstattung, Lage im Büro) sind einsehbar
- [ ] Ausgewählter Arbeitsplatz kann bestätigt werden
- [ ] Bestätigung führt zur nächsten Buchungsseite
