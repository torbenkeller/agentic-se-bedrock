# Bürostandort auswählen

## User Story

Als INNOQ-Mitarbeiter möchte ich einen Bürostandort auswählen, damit ich Räume an meinem gewünschten Standort finden kann.

## Beschreibung

INNOQ verfügt über acht Bürostandorte (Monheim, Berlin, Hamburg, Köln, München, Zürich, Baar, Offenbach). Bevor ein Mitarbeiter einen Konferenzraum buchen kann, muss er zunächst den gewünschten Standort auswählen. Diese Auswahl ist der erste Schritt in der User Journey "Raumbuchung" und bildet die Grundlage für alle weiteren Aktionen wie das Anzeigen verfügbarer Räume oder das Filtern nach Kapazität und Ausstattung.

Die Standortauswahl ist besonders wichtig, da INNOQ-Mitarbeiter häufig remote arbeiten und ihre Bürotage an verschiedenen Standorten planen können. Die klare Strukturierung nach Standorten ermöglicht eine übersichtliche und effiziente Navigation im Multi-Standort-System Calvin.

## Akzeptanzkriterien

- [ ] Alle acht INNOQ-Standorte werden zur Auswahl angeboten (Monheim, Berlin, Hamburg, Köln, München, Zürich, Baar, Offenbach)
- [ ] Der Mitarbeiter kann genau einen Standort auswählen
- [ ] Nach Auswahl eines Standorts wird dieser als aktiver Standort für die weitere Navigation verwendet
- [ ] Der ausgewählte Standort ist für den Mitarbeiter jederzeit erkennbar
- [ ] Die Standortauswahl kann jederzeit geändert werden
