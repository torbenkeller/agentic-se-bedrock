# Verfügbarkeit prüfen

## User Story

Als INNOQ-Mitarbeiter möchte ich die Verfügbarkeit für meinen gewünschten Zeitraum prüfen, damit ich sicher sein kann, dass der Raum frei ist.

## Beschreibung

Nachdem ein INNOQ-Mitarbeiter einen Standort und Konferenzraum ausgewählt sowie Datum und Zeitraum festgelegt hat, möchte er die Verfügbarkeit des Raums für genau diesen Zeitraum prüfen. Diese Funktion ist zentral für das Kernversprechen von Calvin: die Sicherheit, dass ein gebuchter Konferenzraum auch wirklich verfügbar ist.

Die Verfügbarkeitsprüfung gibt dem Mitarbeiter eindeutiges Feedback, ob der gewünschte Konferenzraum im angegebenen Zeitraum frei oder bereits belegt ist. Dies verhindert Doppelbuchungen und gibt dem Mitarbeiter die nötige Transparenz für seine Buchungsentscheidung.

## Akzeptanzkriterien

- [ ] Die Verfügbarkeit eines Konferenzraums kann für einen gewählten Zeitraum geprüft werden
- [ ] Ein freier Raum wird eindeutig als "verfügbar" gekennzeichnet
- [ ] Ein belegter Raum wird eindeutig als "nicht verfügbar" gekennzeichnet
- [ ] Die Prüfung berücksichtigt bereits bestehende Buchungen für den Raum
- [ ] Das Ergebnis der Verfügbarkeitsprüfung wird dem Mitarbeiter sofort angezeigt
- [ ] Bei Verfügbarkeit kann der Mitarbeiter direkt zur Buchung fortfahren
